# Common Errors

| Code | Message |
|:-----:|:-----------:|
| CM0000 | success |
| CM0400 | bad request |
| CM0500 | internal server error |

---

## CM0000

- **Domain**: common
- **Code**: CM0000
- **Message**: success
- **Cause**: operation completed successfully
## CM0400

- **Domain**: common
- **Code**: CM0400
- **Message**: bad request
- **Cause**: invalid input or malformed request
## CM0500

- **Domain**: common
- **Code**: CM0500
- **Message**: internal server error
- **Cause**: unexpected server-side error
