# Payment Errors

| Code | Message |
|:-----:|:-----------:|
| PM0001 | insufficient balance |
| PM0002 | payment gateway timeout |

---

## PM0001

- **Domain**: payment
- **Code**: PM0001
- **Message**: insufficient balance
- **Cause**: user has not enough balance
## PM0002

- **Domain**: payment
- **Code**: PM0002
- **Message**: payment gateway timeout
- **Cause**: no response from payment gateway
